# app/

이 폴더는 아직 비어 있는 **실제 서비스 코드가 들어갈 자리**입니다.

- `async/`, `fast_api/`, `uv_import/`, `db_crud/` 등 다른 폴더들은 토픽별 개념 학습/실습용 코드입니다.
- 이 `app/` 폴더는 그 학습 내용을 기반으로 만드는 **실제 서비스(제품)**를 위한 공간입니다.
- 사용자가 "여기 내용 기반으로 ~하는 기능 추가해줘" 라고 요청하면, 그 기능은 이 `app/` 폴더 안에 구현합니다.

아직 구체적인 서비스 설계는 정해지지 않았으며, 앞으로 기능이 추가될 때마다 이 폴더 아래 구조가 채워질 예정입니다.
