# python-guidebook

## 목적

이 레포는 **토픽별로 폴더를 만들어 파이썬 관련 개념을 학습/실습하는 코드를 모아두는 개인 학습 가이드북**입니다.

- 각 토픽 폴더 안에는 실습 코드(`*.py`)와 해당 개념을 한글로 정리한 문서(`docs/*.md`)가 함께 들어갑니다.
- Claude Code는 이 레포를 기반으로, 사용자가 특정 개념/코드에 대해 질문하면
  1. 먼저 레포 내 관련 코드/문서를 찾아서 답변하고
  2. 레포 내용만으로 부족하면 웹 검색 등을 통해 보충 설명을 제공하는 역할을 합니다.

## 폴더 구조 (토픽별)

- `async/` — 이터레이터, 제너레이터, 코루틴(asyncio) 개념 및 실습
  - `*.py`: 실습 코드
  - `docs/`: 개념 정리 문서 (이터레이터, 제너레이터, 코루틴, 발전과정 등)
- `fast_api/` — FastAPI 실습 서버 (라우터, 스키마, CORS, 워커, Docker 등)
  - `routers/`, `schemas/`: 실습 코드
  - `docs/`: async와 FastAPI, Docker, workers 관련 가이드
- `uv_import/` — uv 프로젝트 구조 및 모듈 import(`__init__.py`, `__name__ == "__main__"`) 학습
  - `main.py` → `service/` → `utils/` 계층 구조 예제
  - `docs/`: uv 설정/실행 관련 가이드
- `db_crud/` — DB CRUD 관련 토픽 (현재는 `docs/` 개념 문서만 있고 코드는 미작성)
- 새 토픽은 위와 같은 패턴(코드 + `docs/` 개념 문서)으로 폴더를 추가해 나갈 예정
- `app/` — 위 토픽 폴더들의 학습 내용을 기반으로 만드는 **실제 서비스(제품)** 코드가 들어갈 폴더. 현재는 빈 상태이며, 아래 규칙에 따라 채워짐

## 작업 규칙: 학습 코드 vs 서비스 코드

- `async/`, `fast_api/`, `uv_import/`, `db_crud/` 등 토픽 폴더 = **개념 학습/실습용** 코드. 원칙적으로 이 폴더들은 학습 목적 실습 코드를 담는 용도이며, 서비스 기능을 여기에 얹지 않음
- `app/` = 그 학습 내용을 바탕으로 만드는 **실제 서비스**. 사용자가 "여기(레포) 내용 기반으로 ~하는 기능 추가해줘" 라고 요청하면, 그 기능 구현은 `app/` 폴더 아래에서 진행함

## 참고

- 프로젝트 관리: `uv` (`pyproject.toml`, `uv.lock`)
- Python `>=3.12`
- 주요 의존성: `fastapi`, `uvicorn`, `psutil`
- 문서(`docs/*.md`)는 한글로 작성됨
