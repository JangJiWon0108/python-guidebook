# 제너레이터 (Generator)

이터레이터를 클래스 없이 함수로 간단하게 만드는 방법.
`yield` 키워드를 사용하는 함수가 제너레이터 함수이고, 호출하면 제너레이터 객체(이터레이터)를 반환한다.

---

## yield 기본 동작

```python
def my_gen():
    print("A 실행")
    yield 1          # 1을 반환하고 여기서 일시정지
    print("B 실행")
    yield 2          # 2를 반환하고 여기서 일시정지
    print("C 실행")
    # 함수가 끝나면 StopIteration 자동 발생


g = my_gen()  # 호출해도 함수 본문이 실행되지 않음. 제너레이터 객체만 반환.

print(next(g))  # "A 실행" 출력 후 1 반환 (yield 1 에서 멈춤)
print(next(g))  # "B 실행" 출력 후 2 반환 (yield 2 에서 멈춤)
print(next(g))  # "C 실행" 출력 후 StopIteration 발생
```

**핵심:** `yield` 는 함수의 현재 상태(지역변수, 실행 위치)를 그대로 보존한 채로 멈춘다.
다음 `next()` 호출 시 멈춘 곳 바로 다음 줄부터 재개한다.

---

## 이터레이터 클래스 vs 제너레이터 비교

```python
# 이터레이터 클래스로 구현 - 번거로움
class CountUp:
    def __init__(self, max):
        self.max = max
        self.current = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.current >= self.max:
            raise StopIteration
        value = self.current
        self.current += 1
        return value


# 제너레이터 함수로 구현 - 훨씬 간단
def count_up(max):
    current = 0
    while current < max:
        yield current   # StopIteration도 자동 처리됨 (함수가 끝나면 자동 발생)
        current += 1


# 둘 다 동일하게 동작
for n in CountUp(3):
    print(n)   # 0, 1, 2

for n in count_up(3):
    print(n)   # 0, 1, 2
```

---

## 제너레이터는 이터레이터다

```python
def my_gen():
    yield 1
    yield 2
    yield 3


g = my_gen()

# 제너레이터는 __iter__와 __next__를 모두 가짐 → 이터레이터
print(iter(g) is g)  # True

# for 루프에 바로 사용 가능
for n in my_gen():
    print(n)  # 1, 2, 3
```

## 무한 시퀀스

이터레이터의 핵심 장점 - 메모리에 전부 올리지 않아도 됨.

```python
def infinite_counter(start=0):
    """끝없이 숫자를 하나씩 만들어내는 제너레이터"""
    n = start
    while True:      # 무한루프지만 메모리 문제 없음
        yield n      # 하나 만들고 멈춤 → 다음 next() 호출 때까지 대기
        n += 1


counter = infinite_counter()
print(next(counter))  # 0
print(next(counter))  # 1
print(next(counter))  # 2
# 필요한 만큼만 꺼내서 사용
```

## 제너레이터 표현식

리스트 컴프리헨션과 문법이 같지만 `[]` 대신 `()` 사용.

```python
# 리스트 컴프리헨션 - 즉시 100만개 생성, 메모리에 전부 올라감
squares_list = [x ** 2 for x in range(1_000_000)]

# 제너레이터 표현식 - 필요할 때 하나씩 계산, 메모리 효율적
squares_gen = (x ** 2 for x in range(1_000_000))

print(next(squares_gen))  # 0
print(next(squares_gen))  # 1
print(next(squares_gen))  # 4
```

## 정리

```
yield 있는 함수    →  제너레이터 함수
제너레이터 함수()  →  제너레이터 객체 (이터레이터)
yield             →  값 반환 + 상태 보존 + 일시정지
next()            →  멈춘 곳에서 재개
함수 끝           →  StopIteration 자동 발생
```

- 클래스 없이 이터레이터를 만들 수 있어 코드가 간결하다.
- 상태(지역변수, 실행 위치)를 자동으로 보존한다.
- 값을 미리 만들어두지 않고 필요할 때 생성하므로 메모리 효율적이다.
